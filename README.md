# FinDash

A modern, minimal finance dashboard built with React, Vite, and Tailwind CSS. I made this to help keep track of personal expenses and income without all the clutter of bloated enterprise apps.

## Features

- **Dashboard Overview:** Quick glance at your total balance, income, and expenses with a responsive area chart showing your balance trend over time.
- **Transactions:** A clean data table supporting search by category or amount, filtering by type, and sorting.
- **Insights:** Visual breakdown of where your money is going, highlighting your highest spending category and monthly comparison trends.
- **Role-based UI:** Switch between 'Admin' (can add/edit/delete transactions) and 'Viewer' (read-only mode).
- **Dark Mode:** Fully supported and persists across reloads.
- **Local Storage:** Everything saves locally in your browser, no backend required.

## Getting Started

1. Clone or download the repo.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Open the local link in your browser.
